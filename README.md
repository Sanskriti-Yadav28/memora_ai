# 🧠 Memora AI – Smart Memory Search & Photo Gallery

Memora AI is a modern, interactive web application designed to help users organize, search, and manage their digital memories efficiently.  
The project demonstrates how a clean user interface combined with intelligent search logic can enhance user experience in photo management systems.

## 📌 Project Overview
With the rapid growth of digital photos, managing and finding specific memories becomes challenging.  
**Memora AI** solves this problem by providing:
- Keyword-based smart memory search  
- Categorized photo gallery  
- Interactive upload simulation with indexing  
- A clean, dashboard-style user interface  

This project is built entirely using **HTML, CSS, and JavaScript**, making it lightweight and easy to deploy using GitHub Pages.

---

## ✨ Key Features
- 🔍 **Smart Search System**  
  Search memories using natural keywords such as *birthday, travel, family,* etc.

- 🗂️ **Gallery with Filters**  
  Photos can be filtered by categories like Outdoor, Family, Celebration, Sports, and Travel.

- ⬆️ **Photo Upload Simulation**  
  Drag-and-drop upload interface with progress animation and indexing simulation.

- 📊 **Dashboard View**  
  Displays memory statistics, recent activity, storage usage, and popular tags.

- 🎨 **Modern UI & UX**  
  Responsive layout, smooth animations, toast notifications, and consistent design system.

---

## 🛠️ Technologies Used
- **HTML5** – Structure of the application  
- **CSS3** – Styling, layout, animations, and responsive design  
- **JavaScript (Vanilla JS)** – Application logic, search scoring, DOM manipulation
